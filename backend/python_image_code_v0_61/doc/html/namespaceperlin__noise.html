def perlin_noise.sample_points (   x,
  t,
  max_x 
)

Private.

Returns a tuple that can be used for cubic interpolation the tuple is the tuple x0, x1, x2, x3, alpha such that 

 x0 = x // t * t
 x1 = x0 + t
 x = alpha * t + x0
 x = x1 - (1 - alpha) * t
 x0 <= x <x1
